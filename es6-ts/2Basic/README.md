## 基本注意点

1.":"后面是类型；
2.ts 只是在编译阶段执行的；
3.编译之后类型就消失了；

## 基本类型

比如：

```ts
let a: number = 100;
```

1.支持基本的类型推导；

```ts
let a = 100;
```

这个地方 a 默认就是 number 类型；

2.基本数据类型使用小写, 一般大写用来描述包装类型
