1.下载nodejs 地址：https://nodejs.org/en/
2.下载成功之后，nodejs和npm都安装好了，因为npm集成在nodejs上（nodejs -version,npm -vesrion）
3.修改nodejs的全局文件位置和缓存文件位置：
	1.在nodejs的安装目录下新建两个文件：node_cache,node_global
	2.打开cmd，执行以下语句npm config set prefix "C:\Program Files\nodejs\node_global（node_global的目录）(这个目录可以先不创建)"
			       npm config set cache "C:\Program Files\nodejs\node_cache（node_cache的目录）"
	3.在系统变量下和用户变量下新建"NODE_PATH"，输入”C:\Program Files\nodejs\node_global\node_modules（node_global的目录）"
	
	4.在环境变量中配置C:\Program Files\nodejs\node_global
4.下载webstorm(略);
